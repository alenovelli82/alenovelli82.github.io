Powershell Menus

This Powershell menu is in production and utilizes the power of PowerShell scripting to assist with various automated tasks related to Active Directory user management, security group management, computer management, and additional tools for processes, applications, files and folders, services, and drivers. The menu is divided into four sections, each with a specific focus.

The first section, AD User Management, includes options for unlocking user accounts, changing passwords, forcing users to change passwords at next login, listing user group memberships, displaying user account information, and performing a wide search. The second section, Security Groups in AD, includes options for searching for groups by name, adding and removing users from groups, and listing group members. The third section, Computers, includes options for retrieving local information, restarting and shutting down computers, sending messages to users, and disabling Juniper Network Services. The fourth section, MORE TOOLS, includes options for managing processes, applications, files and folders, services, drivers, and event logs.

The menu also includes some options for troubleshooting and analyse client logs.
